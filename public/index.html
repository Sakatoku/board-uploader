Board Uploader

ホワイトボード共有アップローダー

Board URL

ボードを準備しています...

ここにファイルやテキストを置けます

ドラッグ&ドロップ、クリップボード貼り付け、または上部のボタンから追加してください。

配置後は他の人が同じ URL で参照し、ファイルをダウンロードできます。

要件定義

  • ホワイトボード上の任意位置にテキスト、画像、各種ファイルを配置できる
  • 画像は貼り付け・アップロード直後にボード上で確認できる
  • 配置結果は共有 URL で他ユーザーが参照できる
  • ファイルは各カードからダウンロードできる
  • MVP としてリンク共有型・認証なしで成立させる

設計

  • Express サーバーが静的 UI と API を提供
  • ボード情報は JSON、バイナリは runtime/uploads に保存
  • 各アイテムは absolute 座標を持ち、ドラッグ終了時に保存
  • URL ごとに 1 ボードを表し、同 URL を開けば同じ内容を表示

使い方

  1. ページを開くと共有ボードが自動生成されます
  2. ファイル追加・ドラッグ&ドロップ・貼り付けでアイテムを登録します
  3. カードのヘッダーをドラッグして好きな位置へ移動します
  4. 共有 URL を渡すと他の人も同じボードを見られます
